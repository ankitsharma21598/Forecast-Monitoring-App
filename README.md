# Forecast Monitoring App

A comprehensive system for monitoring, analyzing, and visualizing wind forecast data. This project combines a TypeScript-based backend, a React frontend for visualization, and a Python-based data analysis module.

## Project Structure

This repository is organized as a monorepo containing the following components:

- **[frontend/](Forecast monitoring app/frontend)**: A React + Vite application for visualizing wind forecast charts and trends.
- **[backend/](Forecast monitoring app/backend)**: An Express.js + TypeScript API that serves wind forecast data.
- **[Data Analysis/](Forecast monitoring app/Data Analysis)**: Python notebooks and research for forecast accuracy and data processing.

## Getting Started

To run the entire project locally, follow the instructions in each subdirectory:

1.  **Backend**: Set up and start the API server.
    - See [backend/README.md](Forecast monitoring app/backend/README.md)
2.  **Frontend**: Start the development server for the web app.
    - See [frontend/README.md](Forecast monitoring app/frontend/README.md)
3.  **Data Analysis**: (Optional) Run the analysis notebooks.
    - See [Data Analysis/README.md](Forecast monitoring app/Data Analysis/README.md)

## Tech Stack

- **Frontend**: React, Recharts, Vite, TypeScript, pnpm.
- **Backend**: Node.js, Express, TypeScript, Axios, Day.js.
- **Analysis**: Python, Jupyter, Pandas, Matplotlib, Seaborn.
